# 场景一致性提示策略

## 场景定义模板

```
场景: [场景名称]
环境: [环境类型]
时间: [时间/季节/天气]
光照: [光照条件]
色调: [主要色调]
氛围: [情感氛围]
视角: [视角描述]
风格: [艺术风格]
细节: [重要细节元素]
```

## 场景一致性示例

### 未来城市场景

```
场景: 新上海2150
环境: 超现代化城市中心
时间: 雨夜
光照: 霓虹灯光和全息投影的蓝紫色照明
色调: 深蓝色、紫色和霓虹粉色
氛围: 神秘而充满科技感
视角: 城市街道视角，高楼大厦环绕
风格: 赛博朋克数字艺术，高对比度
细节: 飞行汽车，全息广告，反光的湿润地面，蒸汽从地下管道升起
```

### 奇幻森林场景

```
场景: 精灵圣林
环境: 古老的魔法森林
时间: 黎明，春季
光照: 柔和的晨光透过树叶形成光束
色调: 翠绿色、金色和淡蓝色
氛围: 神秘而宁静
视角: 森林小径视角，巨大古树环绕
风格: 奇幻插画，细腻的细节
细节: 发光的蘑菇，飘浮的魔法粒子，藤蔓缠绕的石头，小溪流水
```

## 保持场景一致性的技巧

1. **创建场景参考表** - 详细记录场景的所有元素和特征
2. **使用相同的环境描述词** - 每次提示中使用相同的关键环境特征
3. **保持光照条件一致** - 详细描述并在每次提示中重复相同的光照
4. **维持相同的色彩方案** - 使用相同的主色调描述
5. **固定视角和比例** - 如"鸟瞰图"、"近景"或"全景"
6. **参考之前的生成结果** - 如果平台支持，引用之前生成的图片 ID
